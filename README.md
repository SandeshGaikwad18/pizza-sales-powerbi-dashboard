🍕 Pizza Sales Dashboard – Power BI Project
📌 Project Title

Pizza Sales Business Intelligence Dashboard

🎯 Problem Statement

Businesses often struggle to extract meaningful insights from raw sales data.
This project analyzes 48K+ pizza sales transactions to identify revenue trends, customer ordering behavior, and product performance for better data-driven decision-making.

🛠 Tools Used

Power BI Desktop

SQL / MySQL

DAX (Data Analysis Expressions)

Data Cleaning & Transformation

📊 Key KPIs

Total Revenue

Total Orders

Total Quantity Sold

Average Order Value (AOV)

Sales by Category

Sales by Pizza Size

Daily & Monthly Trends

🔎 Key Insights

Total Revenue generated: ₹817K+

Classic category contributes the highest revenue

Weekends (Friday & Saturday) show peak orders

Large size pizzas account for the highest sales share

July and January recorded the highest monthly sales
